
# 5x5 Grid Game

A game where you smash buttons on the terminal to move yourself away from a ghost like enemy in a 5x5 grid.

# Text-Based Ghost Tracker Simulation

## Overview

A terminal-based grid game where the player navigates a map while being chased by an enemy.
The enemy uses basic AI to track and approach the player.

## Features

* Grid-based movement (WASD controls)
* Enemy tracking logic (targeted movement)
* Turn-based system
* Number of Clicks
* Logging system

## Controls

You are expected to type in W/A/S/D in the terminal
* W → Up
* A → Left
* S → Down
* D → Right
Any unknown inputs would be ignored and asked again

## How to Run

Download the .py file and run in any IDE, just make sure python can run in your IDE

## Tech Stack

* Python
* Object-Oriented Programming (OOP)
